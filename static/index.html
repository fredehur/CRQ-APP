The CRQ intelligence pipeline runs across five regional theatres in parallel — collecting open-source signals, routing each region through a triage agent, and producing structured intelligence briefs for any region that escalates. Approved regional briefs are synthesised into a global cross-regional report and rendered into board, CISO, and RSM deliverables. Every handoff between agents is a file on disk. Every output is validated by a separate agent before it is accepted.
12 pipeline nodes 5 regions · parallel fan-out 3 agent models — Haiku · Sonnet · Opus stop-hook validated filesystem state — no shared memory
