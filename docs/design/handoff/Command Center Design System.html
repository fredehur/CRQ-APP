Command Center — Web Design System

The dark-theme visual vocabulary the CRQ Command Center speaks. Primitive component library (Section A), applied shared chrome (Section B), and a bookmark of consumed tokens (Section C). Extends tokens.css; no parallel severity scale, no hard-coded hex, IBM Plex Sans throughout. Severity is semantic-only (pills, dots, chips). Cyan is cyber/accent-only.
